***MESSAGES WITH IDENTIFYING INFORMATION
- Messages with identifying information have children.  
	- .avatar
	- .tstamp
	- .by

This data should always be pulled:
- Message ID from parent.
- Avatar URL from parent->children->.avatar->img.src
- Timestamp from parent->children->.tstamp->innerHTML 
- By from parent->children->.by->innerHTML

****TEXT MESSAGES
IDENTIFY:
	-.message general.  

Similarities: 
- Both are divs of class message general.  
- The text of the message is the parent's outerhtml.  

DATA TO COLLECT
- Content from parent->html 


****ROLL RESULTS
// Output in response to /roll command.
Identifying features:
- class .message.rollresult

Data to collect:
- Formula from parent->children->formula->innerHTML
- Result from parent->children->.rolled.ui-draggable->innerHTML


****ATTACKS
IDENTIFYING INFORMATION
- Parent->first child has class of  .sheet-rolltemplate-atk

DIFFERENCES
- With advantage: parent->.sheet-rolltemplate-atk->child->child->child is '.sheet-solo'
- Without advantage: parent->.sheet-rolltemplate-atk->child->child->child is 'sheet-adv'

DATA TO PULL
- WITH ADVATNAGE: 
	- ROLL1: First child of first sheet.adv's text.
	- ROLL2: First child of second sheet.adv's text. 
	- SUBLABLEL: First child of .sheet-sublabel's text. 
	- ATTACK NAME: First child of .sheet-label's text. 
- WITHOUT ADVANTAGE:
	- ROLL1: .sheet-solo's child's text. 
	- SUBLABLEL: First child of .sheet-sublabel's text. 
	- ATTACK NAME: First child of .sheet-label's text. 

- DESCRIPTIONS: Child of .sheet-desc (IF PRESENT)
